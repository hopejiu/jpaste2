## 修复 QuickLaunch web 窗口唤醒后秒关的 bug

**现象：** 按快捷键唤醒 web 窗口后，窗口立即被隐藏。

**根因：** `start_autohide_timer()` 中用 `mpsc::channel` + `recv_timeout` 实现定时器。当 `Focused(true)` drop sender 取消 timer 时，receiver 返回 `Disconnected` 错误，但原代码用 `is_err()` 判断，把 `Disconnected` 和 `Timeout` 混为一谈，导致 timer 被取消时反而执行了 hide。

**修复：** `lib.rs` 中用 `match` 区分三种情况：
- `Err(Timeout)` — 真正超时，执行 hide
- `Err(Disconnected)` / `Ok(())` — sender 被 drop 或收到信号，不执行 hide

**验证：** 重新构建后窗口正常显示，不再秒关。
